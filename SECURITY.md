# Security Policy

Please read the following:

## Scope

This repository is a **personal configuration project** intended for use on my own computers and personal servers.
It contains scripts, dotfiles, and automation for installing CLI/GUI tools and configuring systems to my personal preferences.

- This project is **not intended for production, enterprise, or shared environments**.

## Usage Disclaimer

- All scripts are provided **as-is**
- They may modify system settings, install software, overwrite configuration files, or change defaults
- Scripts should **only be run after reviewing and testing** in a safe environment

By using anything in this repository, you acknowledge that:
- You understand what the scripts do
- You accept full responsibility for running them
- I am **not responsible for data loss, system breakage, security issues, or misconfiguration**

## Security Considerations

- Secrets, credentials, and machine-specific configuration **should not** be committed
- Users should review scripts before execution
- Some scripts may assume elevated privileges (e.g. `sudo`)

## Reporting Issues

Because this is a personal project:
- Security vulnerabilities do not follow a formal disclosure process
- If you notice something concerning, feel free to open an issue or reach out informally: [Email me](mailto:tafaranyamhunga@proton.me)

No guarantees are made regarding fixes or response times.
